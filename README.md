# Dotfiles

## Installation

Go to your home directory and clone the repo:

```bash
cd ~ && git clone https://github.com/nekhaevskiy/dotfiles.git
```

Symlink everything via:

```bash
~/dotfiles/install.sh
```

or just select what you want manually.
